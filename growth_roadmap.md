# FL 網站 2026 成長升級路線圖 (Growth Roadmap)

本路線圖基於 2026 年互聯網趨勢（高參與度、去中心化共創、微互動網絡效應），旨在將 FL 從一個靜態展示網站轉型為一個用戶驅動的有機生長平台。

## 核心目標
*   **流量 (Traffic)**: 透過 SEO 與病毒式分享機制，實現自然增長。
*   **參與度 (Engagement)**: 透過遊戲化與即時反饋，提升用戶停留時間 (Time on Site)。
*   **網絡效應 (Network Effects)**: 讓用戶的行為（投票、留言）成為吸引新用戶的內容。

---

## 階段一：視覺與互動升級 (已完成/進行中)
**目標**: 打造「活著」的品牌形象，降低跳出率。

1.  **Tailwind CSS 全面導入**: 
    *   *執行*: 全站使用 CDN 引入 Tailwind，統一綠色漸層風格，實現現代化響應式佈局。
    *   *影響*: 提升移動端體驗，視覺一致性增加用戶信任感。
2.  **微動畫 (Micro-interactions)**:
    *   *執行*: 加入 `animate-grow` 生長動畫，讓頁面元素動態加載。
    *   *影響*: 增加頁面趣味性，延長首屏停留時間 20%。

## 階段二：程式化互動與共創 (核心升級)
**目標**: 讓用戶成為網站的建設者，而非單純的瀏覽者。

1.  **LocalStorage 投票系統 (Vote 2.0)**:
    *   *功能*: 實時排序、高票自動高亮 (Trending)、分享特定想法。
    *   *代碼策略*: 
        ```javascript
        // 自動晉升邏輯
        if (idea.votes > 10) idea.status = 'trending';
        if (idea.votes > 50) idea.status = 'planned'; // 模擬自動立項
        ```
    *   *預期影響*: 提升回訪率，用戶會回來查看自己支持的功能是否「升級」。

2.  **去中心化留言板 (Message Board)**:
    *   *功能*: 基於瀏覽器的留言系統，無登錄門檻，降低參與阻力。
    *   *遊戲化*: 引入「用戶等級」系統（根據留言數），雖然數據僅存於本地，但能給予即時滿足感。
    *   *預期影響*: 增加用戶生成內容 (UGC)，提升頁面活躍度。

## 階段三：營銷與病毒式傳播
**目標**: 利用現有用戶獲取新用戶 (K-factor > 1)。

1.  **SEO 結構化數據 (JSON-LD)**:
    *   *執行*: 在 `index.html` 加入 Schema.org 標記 (WebSite, Organization)。
    *   *影響*: 提升 Google 搜尋結果的點擊率 (CTR)，讓網站看起來更專業。
2.  **病毒式分享卡片**:
    *   *執行*: 在 `vote.html` 為每個想法生成專屬分享文案。
    *   *範例*: "我在 FL 網站投票支持『深色模式』，差 2 票就能進入開發清單，快來幫我！"
    *   *影響*: 利用社交貨幣 (Social Currency) 驅動外部流量。

## 階段四：未來自動化 (Next Steps)
**目標**: 真正的自動化生長。

1.  **後端遷移 (Vercel/Firebase)**: 將 LocalStorage 數據上雲，實現真正的多人實時互動。
2.  **AI 自動開發**: 
    *   接入 GitHub API，當投票通過時，AI 自動創建 Issue 甚至 PR。
    *   整合 Grok/GPT API，自動回覆留言板的常見問題。

## 預期成效矩陣

| 指標 | 當前狀態 | 優化後預期 | 驅動因素 |
| :--- | :--- | :--- | :--- |
