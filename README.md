# FL - Free Life

FL (Free Life) is a personal integrated website dedicated to sharing and collecting personal experiences that inspire a liberated lifestyle.

## Features

- **Homepage**: Self-introduction and community rules.
- **Stories**: A collection of user-submitted stories about freedom and growth.
- **Voting**: Interactive voting system with animated freedom birds.
- **Submission**: Form to share your own stories.
- **Responsive Design**: Optimized for both mobile and desktop.

## How to Run

Since this project uses pure HTML, CSS, and JavaScript, you can run it using any simple static file server.

### Option 1: VS Code Live Server
1. Install the "Live Server" extension in VS Code.
2. Right-click on `index.html` and select "Open with Live Server".

### Option 2: Python (if installed)
Open a terminal in the project directory and run:
```bash
python3 -m http.server
```
Then open `http://localhost:8000` in your browser.

### Option 3: Node.js (npx)
If you have Node.js installed:
```bash
npx serve .
```

### Option 4: Direct Open
You can simply double-click `index.html` to open it in your browser, though some features (like LocalStorage or navigation) might behave differently depending on browser security policies for local files.
