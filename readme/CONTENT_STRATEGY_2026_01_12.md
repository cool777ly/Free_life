    *   **步驟**：切換一次深色/淺色模式。

    *   **步驟**：閱讀一篇 Build Log 並找到「技術總結」。
    *   **小目標**：帶走一個可用的知識點。

    *   **步驟**：在 Vote 頁面提交一個新功能提案。
